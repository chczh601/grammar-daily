# 测试指南

## 答案判断功能测试

### 测试目的
验证修复后的答案判断功能能够正确区分正确和错误的答案。

### 测试环境
- 微信小程序开发者工具 或 H5 浏览器
- 已完成用户登录

### 测试步骤

#### 测试用例 1：单选题 - 正确答案

1. 打开小程序，进入"每日练习"页面
2. 找到第一道单选题：
   ```
   题目：I _____ to the cinema yesterday.
   选项：
   A. go
   B. went
   C. have gone
   D. will go
   ```
3. 点击选择 **B. went**（正确答案）
4. 点击"提交答案"按钮

**预期结果：**
- ✅ 显示绿色的"回答正确！"提示
- ✅ 显示正确答案为 "B"
- ✅ 显示题目解析
- ✅ 选项 B 显示绿色对勾图标

#### 测试用例 2：单选题 - 错误答案

1. 点击"下一题"按钮
2. 找到下一道单选题
3. 故意选择一个**错误的选项**
4. 点击"提交答案"按钮

**预期结果：**
- ❌ 显示红色的"回答错误"提示
- ✅ 显示正确答案
- ✅ 显示题目解析
- ❌ 错误选项显示红色叉号图标
- ✅ 正确选项显示绿色对勾图标

#### 测试用例 3：填空题 - 正确答案

1. 继续答题，找到填空题：
   ```
   题目：They _____ (play) basketball now.
   ```
2. 在输入框中输入：**are playing**（正确答案）
3. 点击"提交答案"按钮

**预期结果：**
- ✅ 显示绿色的"回答正确！"提示
- ✅ 显示正确答案为 "are playing"
- ✅ 显示题目解析

#### 测试用例 4：填空题 - 错误答案

1. 点击"下一题"按钮
2. 找到下一道填空题
3. 输入一个**错误的答案**（如 "play"）
4. 点击"提交答案"按钮

**预期结果：**
- ❌ 显示红色的"回答错误"提示
- ✅ 显示正确答案
- ✅ 显示题目解析

#### 测试用例 5：大小写和空格处理

1. 找到一道单选题
2. 确认正确答案（例如 "B"）
3. 选择该选项（例如 "B. went"）
4. 点击"提交答案"按钮

**预期结果：**
- ✅ 系统能够正确识别，显示"回答正确！"
- ✅ 不受大小写影响（内部已转换为小写比较）
- ✅ 不受首尾空格影响（内部已进行 trim 处理）
