## 三、质量提升

### 3.1 知识点覆盖

#### 时态模块细分知识点
1. **一般现在时**（6题）
   - 第三人称单数、客观真理、表将来、主将从现

2. **一般过去时**（6题）
   - 规则/不规则动词、used to do、时态辨析

3. **一般将来时**（6题）
   - will、be going to、主将从现、there be句型

4. **现在进行时**（5题）
   - 正在进行、时间标志、表将来

5. **过去进行时**（5题）
   - when/while从句、同时进行、疑问句

6. **现在完成时**（6题）
   - 对现在的影响、since/for、延续性动词、时态辨析

7. **过去完成时**（3题）
   - by+过去时间、过去的过去、延续性动词

8. **将来完成时**（3题）
   - by+将来时间、完成时态

#### 从句模块细分知识点
1. **名词性从句**（15题）
   - 主语从句（3题）：that/whether/what、it形式主语
   - 宾语从句（3题）：疑问词、陈述语序、if/whether
   - 表语从句（2题）：whether、疑问词
   - 同位语从句（2题）：that、疑问词

2. **定语从句**（15题）
   - 关系代词（8题）：who/whom/whose、which/that、省略、介词+which
   - 关系副词（5题）：when/where/why、选择
   - 限制性与非限制性（2题）：区别和用法

3. **状语从句**（10题）
   - 时间状语从句（3题）：when/while/as、as soon as、not...until
   - 条件状语从句（2题）：if、unless
   - 原因状语从句（2题）：because
   - 让步状语从句（2题）：although
   - 结果状语从句（1题）：such...that

#### 非谓语动词模块细分知识点
1. **不定式**（15题）
   - 作主语（2题）：to do、it形式主语
   - 作宾语（3题）：decide to do、疑问词+to do、否定形式
   - 作宾补（2题）：ask sb to do、使役动词
   - 作定语（2题）：后置定语、不及物动词+介词
   - 作状语（2题）：目的状语、原因状语

2. **动名词**（12题）
   - 作主语（2题）：doing、一般性动作
   - 作宾语（4题）：finish doing、mind doing、介词+doing、be busy doing
   - 作表语（1题）：说明内容
   - 作定语（1题）：表示用途
   - 与不定式的区别（1题）：remember to do/doing

3. **分词**（13题）
   - 现在分词（6题）：作定语、作状语、作宾补
   - 过去分词（6题）：作定语、作状语、作宾补
   - 主被动关系（1题）：选择

#### 虚拟语气模块细分知识点
1. **条件句中的虚拟语气**（15题）
   - 与现在事实相反（5题）：were、could
   - 与过去事实相反（5题）：had done、would have done
   - 与将来事实相反（3题）：should do、were to do
   - 含蓄虚拟条件（2题）：without、but for

2. **名词性从句中的虚拟语气**（10题）
   - wish后的虚拟语气（3题）：与现在/过去/将来相反
   - 表建议、要求、命令的动词（5题）：suggest、advise、insist
   - it is+形容词+that从句（2题）：(should) do

3. **其他虚拟语气**（10题）
   - as if/as though（3题）：与现在/过去相反
   - would rather（2题）：与现在/过去相反
   - if only（2题）：与过去相反
   - it's time（2题）：过去式
   - 综合应用（1题）：混合使用

#### 主谓一致模块细分知识点
1. **语法一致原则**（12题）
   - 单数/复数主语（4题）
   - and连接的主语（3题）：and、both...and...、as well as
   - or连接的主语（3题）：either...or...、neither...nor...、not only...but also...
   - 不定代词作主语（2题）：everyone、nobody

2. **意义一致原则**（10题）
   - 集体名词（3题）：family、team
   - 表示时间、距离、金钱的复数名词（2题）
   - 分数、百分数作主语（2题）
   - the number of与a number of（2题）
   - many a/more than one（1题）

3. **就近一致原则**（8题）
   - there be句型（3题）
   - either...or.../neither...nor...（2题）
   - not only...but also...（2题）
   - or连接的主语（1题）

4. **特殊情况**（5题）
   - 定语从句中的主谓一致（2题）

   - 综合应用（1题）

### 3.2 解析质量提升

每道题目的解析都包含以下内容：

1. **考点说明**：明确指出考查的语法点
2. **解题思路**：说明如何判断正确答案
3. **易错点提示**：指出常见错误和注意事项
4. **拓展知识**：适当补充相关知识点

**示例**：
```
题目：If I _____ you, I would accept the offer.
选项：A. am  B. was  C. were  D. be
答案：C

解析：
本题考查虚拟语气在条件句中的用法。表示与现在事实相反的假设，if从句用过去式，be动词统一用were。
易错点：虚拟语气中，be动词不论主语是什么，都用were，不用was。
```
